# 🐝 꿀템(Ggultem) Backend

Spring Boot 기반 중고마켓 및 커뮤니티 웹 서비스의 백엔드 서버입니다.  
게시판, 댓글, 검색, 관리자 기능 등을 구현하며 실제 서비스 구조를 경험하는 것을 목표로 개발하였습니다.

---

📅 개발 기간: 2026.03 ~ 2026.04 (팀 프로젝트)

---

## 👨‍💻 담당 역할 (Backend)

- 게시판 및 댓글 CRUD 기능 구현
- 계층형 댓글 구조 설계 (부모-자식 관계)
- 논리삭제(Soft Delete) 적용 및 조회 조건 처리
- 검색 기능 개선 (HTML → TEXT 컬럼 분리 및 쿼리 수정)

---

## 💡 핵심 구현

### 1. 계층형 댓글 구조
부모-자식 관계를 기반으로 댓글을 설계하여 대댓글까지 계층적으로 표현할 수 있도록 구현하였습니다.

### 2. 논리삭제(Soft Delete)
데이터를 물리적으로 삭제하지 않고 enabled 값을 변경하여 관리하며, 관리자 페이지에서는 삭제된 데이터도 조회할 수 있도록 구현하였습니다.

### 3. 검색 기능 개선
에디터 사용으로 인해 HTML 형태로 저장된 데이터가 검색되지 않는 문제를 해결하기 위해  
HTML 태그를 제거한 TEXT 컬럼을 생성하고 해당 컬럼 기준으로 검색하도록 개선하였습니다.

---

## 📡 API 예시
- GET /board/list : 게시글 목록 조회
- GET /board/{boardNo} : 게시글 상세 조회
- POST /board : 게시글 등록
- PUT /board/{boardNo} : 게시글 삭제 (논리삭제)
- GET /api/reply/list/{boardNo} : 댓글 목록 조회

---

## 📁 프로젝트 구조

Controller → Service → Repository → DB

- Controller: 클라이언트 요청 처리 및 응답 반환
- Service: 비즈니스 로직 처리
- Repository: 데이터베이스 접근
